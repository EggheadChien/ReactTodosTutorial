# Level 12. 不可變的資料
這是 React 系列教學中的簡易範例 TodosApp


## 本階段目標
1. 使用 React 的 Immutability Helper 改寫 _addTodo, _toggleTodo, _editTodo, _deleteTodo
2. 使用 Immutable.js 再次改寫上述方法
